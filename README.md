# EduNova Academy — Full-Stack Web Application

A full-stack academy management platform with student registration, profile management, program applications, and a complete admin panel for reviewing and managing applications.

## 🔗 Live Demo
[Visit Live Site](https://edunova-production-ad3c.up.railway.app)

## 🛠️ Built With
- PHP (Object-Oriented Programming)
- MySQL
- HTML5 & CSS3
- Bootstrap 5
- Sessions for authentication
- Railway (Hosting & Deployment)

## ✨ Features
- **Student-facing features:**
  - User registration and login with secure sessions
  - Profile creation and editing
  - Program application submission
  - Application status tracking

- **Admin panel features:**
  - Secure admin login
  - Dashboard with live statistics (total users, applications, pending/accepted/rejected counts)
  - Filter applications by status (Pending / Reviewed / Accepted / Rejected)
  - Search applications by name, email or program
  - View full application details in a popup modal
  - Accept, reject, or mark applications as under review with one click

## 🗄️ Database Structure
Built with a relational MySQL database connecting users to their submitted applications, supporting full CRUD (Create, Read, Update, Delete) operations.

## 🎯 Purpose
Built as a full-stack project to demonstrate PHP/MySQL skills, including authentication, CRUD operations, OOP principles, and building a complete two-sided system (user-facing + admin-facing) — closely modeled after real-world SaaS-style applications.
